## תיאור הפרויקט
זהו פרויקט React שמדגים אתר לחיילים משוחררים, שמחפשים עבודה , מעסיקים מעלים משרות והמשתמשים יכולים להשאיר פרטים.  


---

## תכונות עיקריות
- הצגת משרות למשתמשים רגילים ומנהלים
- הרשמה / כניסה עם אימות
- הצגת מועמדים למעסיקים
- פריסה רספונסיבית ועיצוב עם Flowbite React

---

## טכנולוגיות
- React
- Flowbite React
- Redux
- Node.js / Express
